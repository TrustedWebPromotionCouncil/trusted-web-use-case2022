# tw2022-dwn
This is the project for the DWebNode web application.  
Runs in Node.js and written in Typescript.

### requirements
"node" version ">=16".
```commandline
nvm install stable --latest-npm
nvm use 16
```

### setup
```commandline
yarn && yarn build
```

### start server
```commandline
yarn start
```

### test
```commandline
yarn test
```
